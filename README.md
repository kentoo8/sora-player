# Sora Player

Sora動画をローカルでプレビュー・管理するためのビューア。

## 起動

```bash
npm install
npm run dev
```

## 基本操作
- **↑ / ↓**: 動画の切り替え
- **Space**: 再生 / 一時停止

## データ構造
- **設定**: `.env` ファイルの `VIDEOS_DIR` で動画ディレクトリのパスを指定します。
- **アカウント判定**: `account.json` の `name` またはディレクトリ名。
- **ソート**: ULID/メタデータから抽出したタイムスタンプの降順。
